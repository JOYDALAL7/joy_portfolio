# Joy Portfolio

A cyber‑tech themed personal portfolio built with **Next.js**, **Tailwind CSS**, and **React**. It showcases:

- Interactive custom cursor (disabled on mobile)
- Animated skill bars with glowing effects
- Real‑time system monitor stats
- A hidden Snake game for fun
- Dark‑mode theme switcher
- Responsive design for mobile and desktop

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to view the site locally.
